# 1. 启动与配置 (Landing Page)

欢迎使用 Strategy.AI 智能演示文稿生成系统。本章节将指导您完成初始配置并启动项目。

## 1.1 API 密钥配置

在使用任何功能之前，您必须先验证您的身份。

*   **操作位置**：屏幕中央输入框。
*   **操作指南**：
    1.  在带有“Key”图标的输入框中粘贴您的 Google Gemini API Key（以 `AIza` 开头）。
    2.  系统会自动验证密钥格式。验证通过后，边框将变为深蓝色，并解锁下方功能模块。
    3.  如果密钥无效，系统会显示红色警告图标。

> ![截图占位：首页 API Key 输入状态，展示验证通过后的蓝色边框效果]

## 1.2 项目目标设定 (Strategy Engine)

这是生成幻灯片的核心入口。您可以通过文本描述或文件上传来定义演示文稿的目标。

*   **核心目标 (Core Objective)**：
    *   在文本框中输入本次演示文稿的主要目的。例如：“分析 Q3 营收下降原因并提出复苏计划”。
    *   *提示：输入越具体，生成的逻辑越精准。*

*   **上传背景文档 (Upload Brief Docs)**：
    *   点击上传区域或直接拖拽文件（支持 PDF, TXT, MD, DOCX）。
    *   系统将自动读取文件内容作为背景知识库。
    *   *注意：单个文件大小建议不超过 8MB。*

> ![截图占位：左侧 Strategy Engine 模块，展示已输入文字和已上传文件列表的状态]

## 1.3 功能入口选择

首页提供三个主要功能模块的入口：

1.  **Start Logic (启动逻辑生成)**：
    *   基于您输入的目标和文件，从零开始构建一套完整的咨询级演示文稿。
    *   需完成 API Key 验证和目标设定后方可点击。

2.  **Import Visual Deck (导入视觉重绘)**：
    *   如果您已有现成的 PDF 幻灯片，仅需对其进行视觉风格的重绘或分析。
    *   点击上传 PDF 文件即可进入。

3.  **Project Iron (文档重构工具)**：
    *   专门用于将扫描版或不可编辑的 PDF 转换为可编辑的 PPTX 文件。
    *   点击最右侧的琥珀色卡片进入独立工具页面。
